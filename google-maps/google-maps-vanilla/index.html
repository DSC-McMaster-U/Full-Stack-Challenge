The purpose of this app was to implement Firebase and interact with the Google Maps API!

You will notice that all there really is to this is a search bar, and the original spirit of the challenge was to enter a longitude and latitude as an input parameter and place a marker on the map for that location.
By leveraging the APIs geocoding feature, we can actually search for any Country, City, Town, business, or co-ordinates and be able to find the location on the map!

You will notice that after you login, several markers are already populated on the map. Every time the app is opened, all the stored markers are drawn on the map, and any new searches will be added to the database as well.

Lastly, if you look around the map at at the current existing markers, you'll find some of my favourite spots that I've traveled to :)
